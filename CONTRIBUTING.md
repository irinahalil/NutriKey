# Contributing

Спасибо за интерес к проекту.

## Локальный запуск

1. Откройте `index.html` в браузере  
или
2. Запустите локальный сервер:

```bash
python3 -m http.server 8000
```

## Правила изменений

- Соблюдайте семантическую HTML-структуру.
- Сохраняйте спокойный визуальный стиль и читаемость.
- Не добавляйте лишние зависимости без необходимости.
- Проверяйте ссылки Telegram после правок.

## Проверка перед коммитом

- Откройте страницу на десктопе и мобильной ширине.
- Убедитесь, что кнопки и CTA работают.
- Проверьте, что изображение `photo.png` отображается корректно.
