# 🥗 Ирина Халил — Landing Page нутрициолога

Современный одностраничный сайт для эксперта в сфере питания.  
Фокус на ясности, доверии и простоте восприятия.

---

## ✨ О проекте

Лендинг разработан для презентации услуг нутрициолога и выстраивания доверия через понятную структуру и спокойный визуальный стиль.

Основная идея — показать, что питание может быть:
- без крайностей
- без перегруза
- с опорой на систему

---

## 🎯 Цели проекта

- Донести экспертность через простой язык
- Сформировать ощущение спокойствия и уверенности
- Привести пользователя к консультации
- Упростить принятие решения

---

## 🧠 Особенности

- Чистая семантическая HTML-структура
- Акцент на читаемость и UX
- Минималистичный и "дышащий" layout
- Контент-ориентированный подход

## 🚀 Запуск проекта

```bash
# вариант 1: просто открыть файл
index.html
```

```bash
# вариант 2: локальный сервер
python3 -m http.server 8000
```

После запуска сервера откройте: [http://localhost:8000](http://localhost:8000)
